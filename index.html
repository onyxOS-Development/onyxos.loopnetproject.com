OnyxOS is currently only a concept and we hope to achieve something beyond the concept phase to create something truly unique
A contribution may work differently than on other projects. As the components can be changed while compiling, the components are working as patches rather than a full source code pull. The patch sources are downloaded, integrated and indexed into the source code and get it's own signature. While compiling caching is going to be enabled to speed up the installation process. The hash from every file is going to be checked as its going to be compiled. When the hash doesn't match up with the indexed one, its cache will be updated by processing the change.
The Atomic App or the device itself uses its update function to download a package, verify it's signature and meta against its integrated one, and reboots into update mode, where the device is going to flash the firmware without the possibility of data loss. When an external flash chip is hooked up, the automatic recovery function will become available as well.
The Atomic App gives you an Interface where you can manage your Device with simplicity. You can easily install and update the firmware, add or delete components, change configuration, explore the filesystem and even access a shell. If you have it connected to a network using cellular functionality, you can also ring it, locate it and deactivate it.
